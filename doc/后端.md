# nvr-server

打包
```sh
docker run -it --name=pyOpenlistSync -v D:\github:/app python:3.11-alpine3.20 /bin/sh
```

```sh
apk add binutils
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
pip install pyinstaller
cd /app/OpenlistSync
pip install --no-cache-dir -r requirements.txt
pyinstaller -n OpenlistSync -F main.py
```

打镜像

```sh
docker build -t syscc/openlistsync:latest .
```

运行命令

```sh
docker run -d -p 8023:8023 -v D:\opt\app\OpenlistSync:/app/data --name=OpenlistSync syscc/openlistsync:latest
```
